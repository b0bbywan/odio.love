No. odio installs on top of an existing Raspberry Pi OS Lite (Trixie) with a [single curl install command](#get-started). You keep your SSH keys, your hostname, and anything else you already run on the Pi. [Pre-built Raspberry Pi Imager images](#get-started) are available if you prefer to start fresh.
